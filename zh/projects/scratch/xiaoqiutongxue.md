``` text [操作说明]
欢迎使用由 @Little-Earthworm 原创《小蚯同学》，以下是小蚯同学的自述
————————————————————————
按键 ....................................................................... 功能
-------------------------------------------------------------------------
[空格] ..................................................................... 对话
-------------------------------------------------------------------------
[T] .......................................................................... 翻译

[A] ........................................................... 中文翻译英文
[B] ........................................................... 英文翻译中文

[回车 / Enter] .................................................. 结束翻译
-------------------------------------------------------------------------
[S] ................................................................... 学习模式
-------------------------------------------------------------------------
[V] ................................................................... 声音设置

[A] .......................................................................... 中音
[B] ...................................................................... 男高音
[C] .......................................................................... 尖细
[D] .......................................................................... 巨人
————————————————————————
我会什么？
我会打招呼；识别（系统）时间、日期、姓名等；翻译（中文 / 英文）、播放（暂停 / 换）音乐等
————————————————————————
翻译
我已经拥有翻译功能啦！目前支持 [中文翻译英文] 和 [英文翻译中文]，按 [T] 或在对话框内输入 [翻译] 或 [T] 即可进入；在对话框里输入 [中文翻译] 可直接进入 [中文翻译英文] 模式，输入 [英文翻译] 可进入 [英文翻译中文] 模式！
————————————————————————
学习模式
学习模式可按 [S] 或在对话框（按 [空格] 键）内输入 [S] 或 [学习模式] 或者在对话框内输入我不知的的东西后即可启动进入后请输入需要学习的问题（如果是对话模式，将直接计入问题，无需输入）以及相应答案，然后进入对话框输入相应问题后，将回答您输入的答案
我可以无限次数学习，但是如果您刷新网页，将全部删除哦！
————————————————————————
声音设置
[声音设置] ，可按 [V] 或在对话框（按 [空格] 键）内输入 [V] 或 [设置] 进入，更改我的声音
————————————————————————
更多功能正在努力研制，欢迎在评论区投稿，让我拥有更强大的功能哦！
```
