``` text [操作说明]
Welcome to Earth OS (1.0) ʙᴇᴛᴀ !

2️⃣ Change decimals to Fractions:
Click the blue button to input the numbers.

When you input a big decimal in it, it will load very slowly (like if you input 0.142857, it may loading about 2h...), but it won't get stuck. Just wait.
----------------------------------------------------------------------
3️⃣ Logarithm calculator:
Click the [log (In / Ib / Ig) ] button to change the type.
Click the other blue button to input the numbers.

This one use the base changing formula for logarithms.

Logarithm is to calculate the exponent;
Antilogarithm is the answer of power;
"lb" means the base number is 2;
"㏑" means the base number is e;
"lg" means the base number is 10;

e.g. ㏒₂8=lb8=3 (2³=8)
```
